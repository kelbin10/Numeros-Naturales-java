# Números Naturales en Java 

Este programa en Java imprime los primeros 10 números naturales utilizando un bucle `for`.

---

 Código

```java
public class NumerosNaturales {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
